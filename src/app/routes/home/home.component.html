Components/h3>

The component is a controller class with a template which mainly deals with a view of the application and logic on the page

Modules

A module can be a library for another module. For instance, the angular2/core library which is a primary Angular library module will be imported by another component.

Services

Services are JavaScript functions that are responsible for doing a specific task only. Angular services are injected using Dependency Injection mechanism and include the value, function or feature which is required by the application
